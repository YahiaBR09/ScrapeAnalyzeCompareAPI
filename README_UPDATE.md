## 🔧 الخوارزميات المستخدمة

### للمقارنة:
1. **Levenshtein Distance**: قياس تشابه النصوص
2. **Jaccard Similarity**: مقارنة مجموعات الكلمات
3. **Weighted Scoring**: توازن ذكي (55% اسم + 25% ماركة + 20% وزن)

### لتنظيف البيانات:
- إزالة الأرقام والوحدات
- إزالة الرموز والعلامات الخاصة
- إزالة الكلمات غير المهمة
